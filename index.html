Selected Projects

  • Learning to Transfer via Modeling Multi-level Task Dependency

    Advisor: Prof. Yizhou Sun (UCLA)

    We construct attention based tasks dependency relationships on both general level and data-specific level, which is used to guide what knowledge should be transferred. Our framework outperforms SOTA multi-task learning methods on “discrete” data (graph and text). And this project also received UCLA-CSST Outstanding Research and Presentation Skills Award (Top 10%).

  • Development of AI Front-view Advanced Driver Assistance System (ADAS)

    Advisor: Dr. Laiye Bi and Dr. Longtao Yuan (Leadmove Technology)

    I independently design an image post-processing system and a camera calibration system. The image post-processing system integrates distortion correction, (inverse) perspective mapping, position detection, and speed measurement. And the camera calibration system is based on Optical Flow and Newton’s Method which improves the accuracy and efficiency of online camera calibration compared to existing methods.

  • GPU Accelerated Mitosis Cell Detection

    Advisor: Prof. Steven Lumetta (UIUC)

    [code] [slides] We utilize SOTA Convolutional Neural Network (CNN) to detect mitosis cells. Further, we optimize the CUDA implementation of CNN with novel fast matrix multiplication algorithm, which accelerates the forward propagation of CNN by 181% compared to existing CUDA implementation.

  • Linux System Design

    Advisor: Prof. Zbigniew Kalbarczyk and Prof. Jian Huang (UIUC)

    [code] We develope the core of an operating system from scratch. We realize several basic functionalities such as system calls, scheduling, and file system. Further, we add several advanced functionalities such as graphical terminal, Chinese Pinyin input method editor, drivers (sound, mouse, tux), ACPI power-off and reboot, and CMOS date time. This project ranked No. 1 among 54 groups in the design competition.
